R.A.G. (Read & Grow) is a book list app built with Python.

  • R.A.G have the following options for the user to interact with:
  • Introductory Message about R.A.G : Greetings message and a brief overview of the app features.
  • Book Menu( view the books list): Display a menu with options of the books that the user can choose from & the book description.
  • User record : The user can register in the program & can mark the books as read or unread to track his progress.
  • Random recommendation : we will recommend a random book to the user based on his preference.
  • Keyword Search : the user can search through the book list through keywords.
